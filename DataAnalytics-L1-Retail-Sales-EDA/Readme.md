# 📊 Retail Sales Data Analysis

## About This Project

Hi! 👋

This is my Retail Sales Data Analysis project, which I completed as part of my **Data Analytics Internship**.

The goal of this project was to explore a retail sales dataset and understand customer behavior, sales trends, and product performance using Python. Through this project, I practiced data analysis, visualization, and drawing business insights from data.

---

## Tools I Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset

The dataset contains retail sales information such as:

- Transaction ID
- Date
- Customer Age
- Gender
- Product Category
- Quantity Purchased
- Price per Unit
- Total Amount
