# Titanic Data Cleaning Project

## 📌 Project Overview
This project demonstrates data cleaning techniques on the Titanic dataset using Python and Pandas.

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## 📋 Tasks Performed
- Data Quality Assessment
- Missing Value Handling
- Duplicate Removal
- Data Standardization
- Outlier Detection (IQR Method)
- Data Type Correction
- Before vs After Comparison
- Exported Cleaned Dataset

## 📂 Files
- Titanic_Data_Cleaning.ipynb
- cleaned_titanic.csv

## 🎯 Outcome
A clean and analysis-ready Titanic dataset suitable for Exploratory Data Analysis (EDA) and Machine Learning.
